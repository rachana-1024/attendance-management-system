\# Attendance Management System



\## Overview

The \*\*Attendance Management System\*\* is a web application used to track employee attendance and working hours. Employees can record their daily check-in and check-out times, and the system stores attendance records for monitoring and reference.



\---



\## Features

\- Employee login

\- Check-in and check-out tracking

\- Automatic working hours calculation

\- View attendance records



\---



\## Technologies Used

\- Frontend: HTML, CSS  

\- Backend: Python (Flask)  

\- Database: SQLite / MySQL  

\- Version Control: Git  



\---



\## How It Works

1\. Employee logs in.

2\. Marks \*\*Check-In\*\* at the start of the day.

3\. Marks \*\*Check-Out\*\* at the end of the day.

4\. The system calculates total working hours and saves the attendance record.
